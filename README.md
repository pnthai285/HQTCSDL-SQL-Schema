# Các lưu ý:
* Hiện tại trong file `data.sql` chỉ mới tạo dữ liệu cho Thái, Thắng, Toàn thôi.
* Ae muốn tạo thêm dữ liệu thì *Append* file `data.sql`, nếu chạy **không lỗi** thì *pull request* lên github nha.
* Ae tạo file `procedures-<tên ae>.sql` để viết procedures của mình nha, kèm theo một `file markdown` để hướng dẫn cách chạy. Khi nào ae xong hết, mình gộp lại rồi chạy xem có lỗi không.
